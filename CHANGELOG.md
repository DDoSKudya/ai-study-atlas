# Changelog

Все заметные изменения проекта фиксируются в этом файле.

## 0.1.1 (2026-04-29)

### Изменено

- `README.md` расширен: подробнее раскрыты идея проекта, философия, структура базы, процесс работы и роль AI/prompt-пайплайна.
- В `README.md` и `INDEX.md` явно зафиксировано, что проект ориентирован на русскоязычного пользователя.
- `INDEX.md` переведен в более операционный формат: добавлен табличный реестр направлений и треков с шаблоном для расширения.
- Репозиторные документы упрощены под практическое ведение базы без лишней бюрократии.

### Исправлено

- Исправлена совместимость `lychee`-конфига с версией `v0.23.x` (параметры в `.lychee.toml`).
- Унифицированы ссылки на глобальные планы: в учебных материалах используется единое имя `mastery_plan.md` вместо старых вариантов (`*_mastery_plan.md`).

### Контент

- В базу добавлен и подключен новый трек: `directions/cs_fundamentals/clean_code`.
- Расширено покрытие учебного контента в направлениях и `pact`-разделах с сохранением единой навигации и naming-конвенции.

## 0.1.0 (2026-04-29)

### Добавлено

- Базовая навигация по проекту: `README.md`, `INDEX.md`.
- Базовые правила ведения контента: `docs/content_contract.md`, `docs/naming_convention.md`.
- Корневой `CHANGELOG.md` как единый журнал изменений.
- Минимальный комплект репозиторных документов для публикации на GitHub (`LICENSE`, `CONTRIBUTING.md`, `SECURITY.md`, шаблоны issue/PR, `.gitignore`).
- Конфигурация `.lychee.toml` и workflow `.github/workflows/markdown-links.yml` для автоматической проверки битых Markdown-ссылок в GitHub Actions.
