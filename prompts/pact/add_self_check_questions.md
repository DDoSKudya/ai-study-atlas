# Add Self Check Questions

**Какой файл имеется в виду:** тот, который был **создан или последним доработан в этом чате** как учебная часть по указанной теме. Не проси пользователя указывать путь — он определяется контекстом чата. Если файлов несколько — кратко уточни, какой дорабатывать.

Нужно добавить в этот файл вопросы для самопроверки по каждому пункту и подпункту, включая маленькие блоки.

Требования:
1. Пройтись по всему содержимому файла.
2. Для каждого смыслового пункта, раздела, подпункта и важного подблока добавить по 2–3 вопроса на понимание (прямые, на сравнение, на причинно-следственные связи, на практическое применение).
3. Не пропускать маленькие пункты, если в них есть идея, термин, правило, ограничение или вывод.
4. Добавлять вопросы прямо в файл, в соответствующие места (не только в конец).
5. Ответы — в формате `<details><summary>Ответ</summary>...</details>`, достаточно полные для повторения материала.
6. Вопросы осмысленные, не шаблонные; учитывать содержание раздела.

Если вопросы уже есть — проверить качество и количество; при необходимости дополнить. Финальный результат вносится в тот же файл из контекста чата.
