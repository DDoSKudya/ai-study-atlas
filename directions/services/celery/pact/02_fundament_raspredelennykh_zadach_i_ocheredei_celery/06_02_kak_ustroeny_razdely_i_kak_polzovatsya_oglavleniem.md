## Как устроены разделы и как пользоваться оглавлением

Внутри каждого основного раздела используется учебный маршрут:

| Блок | Зачем нужен |
| --- | --- |
| **Цель раздела** | Понять результат чтения. |
| **В этом разделе главное** | Собрать 3–7 опорных идей. |
| **Термины** | Зафиксировать словарь. |
| **Теория и правила** | Дать формулировки и инженерные правила. |
| **Пошагово** | Потренировать мышление процессом. |
| **Простыми словами** | Перевести на человеческий язык. |
| **Картинка в голове** | Сделать ментальную модель визуальной. |
| **Как запомнить** | Сжать в формулу. |
| **Примеры** | Увидеть идею в реальных сценариях. |
| **Практика / реальные сценарии** | Узнать, где это встречается. |
| **Типичные ошибки** | Подсветить грабли заранее. |
| **Что будет, если...** | Разобрать последствия ошибок. |
| **Проверь себя** | Вопросы на понимание с ответами под спойлером. |
| **Запомните** | Закрепить итог. |

Оглавление по этапам позволяет читать сверху вниз или возвращаться к конкретной теме после инцидента.
