## Как устроены разделы и как пользоваться оглавлением

Каждый раздел ниже по возможности следует одному шаблону: **цель → главное → термины → теория → пошагово → простыми словами → метафора → как запомнить → примеры → практика → ошибки → последствия → проверка → краткий итог**.

Оглавление сгруппировано по этапам: можно учиться сверху вниз или прыгать по ссылке как по справочнику.
