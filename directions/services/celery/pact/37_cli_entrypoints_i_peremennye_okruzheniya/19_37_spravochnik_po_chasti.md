## Справочник по части

| Категория | Что держать в голове |
|---|---|
| Глобальные флаги | Сначала `-A` и контекст (`--config`, `--workdir`), потом подкоманда |
| Worker | pool/concurrency/queues/limits должны быть согласованы |
| Beat | один активный scheduler и контролируемый state |
| Events | включать осознанно, с учетом стоимости и безопасности |
| Inspect/Control | сначала наблюдение, затем адресное действие |
| Env | гибкость + риск неявных переопределений, нужна политика |
| Process manager | корректные сигналы, таймауты и restart-логика обязательны |
