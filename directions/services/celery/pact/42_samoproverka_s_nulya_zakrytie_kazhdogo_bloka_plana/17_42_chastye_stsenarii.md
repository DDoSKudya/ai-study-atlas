## Частые сценарии

| Ситуация | Как использовать часть 42 |
| -------- | ---------------------------- |
| «Прочитал, но ничего не помню» | Сначала **42.1** на последней прочитанной главе без подглядывания. |
| «Много теории, мало рук» | Открыть **матрицу 42.2** и брать лабы чередованием «любимая/ненавистная тема». |
| «Готов к продакшену?» | Пройти **42.3** целиком + **42.4** устно. |
| «Готовлюсь к собеседованию» | **42.4** + выборочно лабы из **8, 9, 28, 29**. |
| «Не знаю с чего начать после главы N» | **42.1** на этой же главе за 20–30 минут, затем одна строка из [журнала лаб](#labs-progress-template) с датой. |
| «Выгорел, бросаю курс на середине матрицы» | Не догонять номера подряд: [компас слоёв](#labs-compass) — закрыть по **одной** лабе из слоя, где больше всего страха (часто B или C). |

#### Проверь себя: частые сценарии

1. Почему для «ничего не помню» указан **42.1**, а не сразу экзамен **42.3**?
2. В чём разница маршрутов **«готов к продакшену»** и **«собеседование»** из таблицы?
3. Зачем при выгорании предлагается **компас слоёв**, а не «продолжай с того номера, где бросил»?

<details><summary>Ответ</summary>

1. **42.1** восстанавливает **опорные артефакты** на одной главе; **42.3** без них часто превращается в хаотичный проект без модели.
2. Прод требует **склейки** (42.3+42.4); собеседование — **устные якоря** и выборочные лабы 8/9/28/29 без полного экзамена.
3. Страх чаще **слойный**, а не номерной; одна лаба из «страшного» слоя даёт прогресс без ощущения бесконечного долга.

</details>
