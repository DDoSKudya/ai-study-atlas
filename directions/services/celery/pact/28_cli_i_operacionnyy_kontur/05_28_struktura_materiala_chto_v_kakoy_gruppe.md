## Структура материала (что в какой группе)

| Группа разделов | О чем раздел | Зачем изучать |
|---|---|---|
| **Карта части** | соответствие подпунктам `28.1-28.5` | контроль полноты по глобальному плану |
| **Worker CLI** | запуск, флаги, производительность, ошибки | управлять исполнением задач осознанно |
| **Beat CLI** | scheduler, state file, HA-подходы | избежать дублей и провалов расписания |
| **Inspect/Control/Events** | диагностика и remote operations | ускорить triage и инцидентное управление |
| **Утилиты обслуживания** | purge/revoke/multi/report | безопасно выполнять аварийные операции |
| **Безопасность и автоматизация** | RBAC, runbook, CI/CD ограничения | уменьшить риск операционных ошибок |
| **Финал** | справочник, сценарии, вопросы, ошибки, резюме | закрепить ментальную модель |
