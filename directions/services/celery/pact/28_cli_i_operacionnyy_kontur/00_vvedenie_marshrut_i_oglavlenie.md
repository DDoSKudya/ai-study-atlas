# Часть 28. CLI и операционный контур Celery

> **Цель этой части** — превратить работу с Celery CLI из набора «магических команд» в понятный и безопасный операционный инструмент: для запуска, диагностики, управления и аварийных действий в production.
>
> **Ключевая мысль:** команда `celery` — это не просто запуск worker. Это целый «пульт управления» системой фоновых задач, и ошибка в одной команде может стоить вам либо простоя, либо потери задач.
