# Часть IV. Современный синтаксис Python 3

> **Цель:** уверенно использовать современный синтаксис Python 3: f-строки, распаковку, match/case, морж (:=), нововведения 3.10–3.13.

**Краткая шпаргалка (суть):**

| Конструкция | Синтаксис | Назначение |
|-------------|-----------|------------|
| f-string | `f"{x}"`, `f"{x:.2f}"`, `f'{x=}'` | Подстановка в строку; формат; отладка |
| Распаковка | `a, *rest = seq`, `f(*args, **kwargs)` | Несколько имён из одной последовательности; развернуть в аргументы |
| match/case | `match x: case 1: ... case _:` | Сопоставление с образцом вместо if/elif |
| Морж | `if (n := len(data)) > 0:` | Присваивание в выражении |

---

## Маршрут изучения

**Что вы изучите и в каком порядке.** Сначала — **основной синтаксис** (§16–§19): **f-строки** (подстановка в строку, формат, отладка), **распаковка** (присваивание из последовательностей, **\*args**/**\*\*kwargs**), **match/case** (сопоставление с образцом), **морж** (**:=** — присваивание в выражении). Разделы можно идти по порядку. Дальше — **нововведения по версиям** (§19a–§19f): **Union \|**, **zip(strict)**, **ExceptionGroup**, **Self**, **type** statement и др. (3.10–3.13). В конце части — справочник (сценарии «задача → решение», таблица выбора конструкций, терминология) и проверка знаний (вопросы, задания, типичные ошибки, отладка) для повторения и самопроверки.

**Что желательно знать заранее:** Часть I (отличия Py3 от Py2), базовый синтаксис и типы.
