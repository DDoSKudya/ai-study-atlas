# Часть 0. Философия, границы и ментальные модели

> **Цель:** понять, *почему* Python устроен именно так и *как* на нём думать. Можно изучать параллельно с Частью I.

**Краткая шпаргалка (суть):**

| Понятие | Суть |
|---------|------|
| Zen of Python | 19 принципов дизайна; `import this` |
| PEP | Python Enhancement Proposal — процесс изменений языка |
| EAFP vs LBYL | «Проще просить прощения» vs «сначала проверь» |
| Duck typing | Поведение важнее типа; протоколы |
| Мутабельность | list, dict, set — изменяемы; tuple, str, frozenset — нет |

---

## Маршрут изучения

**Что вы изучите и в каком порядке.** Сначала — **история и философия** (0.1): откуда взялись принципы Python (**Zen of Python**, **PEP**), ключевые решения (**GIL**, batteries included) и как язык развивается (Py2 vs Py3). Это даёт контекст «почему так сделано». Дальше — **границы языка** (0.2): когда Python подходит, когда — не лучший выбор, как он встраивается в стек (subprocess, FFI, Cython) и что важно в экосистеме (PyPI, venv). Затем — **ментальные модели** (0.3): как думать при написании кода — «всё объект», **name binding**, мутабельность, **EAFP** vs **LBYL**, **duck typing**. В конце части — справочник: терминология, вопросы, задания; им удобно пользоваться для повторения и самопроверки.

**Что желательно знать заранее:** часть можно изучать параллельно с Частью I (отличия Python 3 от 2.7); достаточно базового знакомства с Python.

## Как устроены разделы и оглавление

В каждом разделе (0.1.1–0.3.8) материал идёт по одному шаблону: **цель** (что вы поймёте) → **термины** (Zen, PEP, EAFP, duck typing и т.д., с кратким определением) → **правила и примеры** (как устроено, как применять) → **граничные случаи** (где легко ошибиться) → блок **«Запомните»** (краткий вывод). Термины в тексте сохранены; в начале раздела или в таблицах они расшифрованы простыми словами.

**Оглавление** разбито на этапы изучения. Каждая строка таблицы — раздел: по ссылке переходите к материалу. В столбце «Содержание» — о чём раздел; в «Ключевые понятия» — термины, которые нужно знать после изучения. Идите по этапам сверху вниз; справочник в конце — для повторения и самопроверки.
