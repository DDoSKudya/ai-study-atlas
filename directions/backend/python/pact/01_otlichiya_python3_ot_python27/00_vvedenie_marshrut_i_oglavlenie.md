# Часть I. Отличия Python 3 от Python 2.7

> **Цель:** разобрать ключевые отличия Python 3 от Python 2.7 в синтаксисе, типах, итераторах, генераторах, функциях и классах (§1–§5).

**Краткая шпаргалка (суть):**

| Область   | Py2 → Py3            | Ключевое изменение             |
| --------- | -------------------- | ------------------------------ |
| print     | Инструкция → функция | `print("x")` с скобками        |
| str/bytes | str = байты, unicode | str = Unicode, bytes — байты   |
| Деление   | `/` целое при int    | `/` всегда float, `//` — целое |
| range     | list                 | Итератор (ленивый)             |
| dict      | .keys() list         | .keys() view (как set)         |
| Итераторы | .next()              | .**next**(), next()            |

---

## Маршрут изучения

**Что вы изучите и в каком порядке.** Сначала — **синтаксис** (§1): как изменились базовые конструкции в Python 3 — **print** как функция, **str** и **bytes**, деление **/** и **//**, **raise**. Затем — **типы** (§2): **range** как итератор, **dict** с view вместо списков, **list**, **set**, **tuple** и их отличия от Py2. Дальше — **итераторы и генераторы** (§3): **next()**, **yield**, протокол итератора, асинхронные генераторы. После — **функции** (§4): объявление, **LEGB**, встроенные (**builtins**). В конце блока — **классы и ООП** (§5): базовые механизмы, декораторы методов, **dunder**, дескрипторы, ****slots****, метаклассы. В конце части — справочник (терминология, вопросы, задания) для повторения и самопроверки.

**Что желательно знать заранее:** базовое знакомство с Python 2 или с любым Python — тогда отличия будут нагляднее.

## Как устроены разделы и оглавление

В каждом разделе (§1–§5) материал идёт по одному шаблону: **цель** (что изменилось в Py3) → **термины** (print как функция, str/bytes, view и т.д., с кратким определением) → **синтаксис и правила** (как писать в Py3) → **примеры** (код Py2 vs Py3) → **граничные случаи** (подводные камни) → блок **«Запомните»** (краткий вывод). Термины в тексте сохранены (**LEGB**, **view**, протокол итератора и т.д.); в начале раздела или в шпаргалке они расшифрованы.

**Оглавление** разбито на этапы изучения. Каждая строка таблицы — раздел: по ссылке переходите к материалу. В столбце «Содержание» — о чём раздел; в «Ключевые понятия» — термины, которые нужно знать после изучения. Идите по этапам сверху вниз; справочник в конце — для повторения и самопроверки.
